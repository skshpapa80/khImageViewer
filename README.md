# khImageViewer

델파이로 만든 이미지 뷰어
델파이 커뮤니티 버전으로 개발했습니다. 

폴더를 클릭하면 썸네일 이미지 리스트를 만드는 데 
그 썸네일 이미지 리스트는 스레드를 사용했습니다. 

트리 모양은 ShellTreeview 를 사용했구요.
